# 当我们说跨性别时，我们在说什么/什么是跨性别？

当我们说“跨性别”时，我们实际上说的是那些指派性别与其性别认同所不相符的个体，是一种先天性的生理错位状态。虽然“跨性别”这一词汇在上世纪才被创造并用于描述这一群体，但无论古今中外，跨性别者始终在世界范围内的各种文化中存在着。跨性别是一个伞式术语，其中包括跨性别女性，指那些指派性别为男性，但性别认同为女性，本质上是女性的个体、跨性别男性，指那些指派性别为女性，但性别认同为男性，本质上是男性的个体，与非二元性别，指那些性别认同不符合传统二元性别划分的个体，是非常规性别者的总称。一些非二元性别者会认为自己属于流体性别，即认为性别是流动的，或是性别酷儿，即那些不符合传统性别规范的个体，而一些非二元性别者也会认为自己不属于跨性别群体，这是完全因人而异的。对于更进一步的说法，我们可以将所有指派性别为男性但是性别认同为女性或者偏向女性的个体统称为“女性倾向跨性别者”，这包含了跨性别女性与非二元性别的情况，如果是指派性别为女性，但是性别认同为男性或者偏向男性的个体，我们则可以将其统称为“男性倾向跨性别者”，同样包括了跨性别男性与非二元性别的情况。当跨性别者持续的对自己的第一性征与第二性征感到厌恶，或对自己的性别角色长期表示不满并因此产生了焦虑情绪，那么就是性别焦虑的情况，跨性别者通常会通过寻求性别肯定序列医疗与真实生活体验等方式来缓解性别焦虑并进行性别过渡。我们知道，一个人具有何种性别并不由其指派性别决定，而是由其性别认同所决定，因此将跨性别女性称作“想要变成女性的男性”是不合适的，我们可以说，跨性别女性本来就是女性，但是遗憾地指派性别没有与其性别认同所相符，所以具有了男性化的身体，接受性别肯定序列医疗对跨性别女性来说是“使身体回到本来就应该是的样子”而非“变成另一种性别”，跨性别男性反之同理。跨性别不意味着一个人应具有什么样的性取向和性别表达，这与指派性别和性别认同是完全不同的概念，无法相互决定与影响。社会上曾使用“变性人”与“性别认同障碍”来描述跨性别者，但这种说法已被国内跨性别社群所弃用，对于前者，在当今中文语境下已成为蔑称，而对于后者，也已被“性别不一致”的说法所取代，使用已过时的称呼是一种极为不尊重的做法并且还会为跨性别群体带来巨大的污名化。

# 为什么会有跨性别的情况/跨性别的成因是什么？

正如前文中提到的，跨性别是持续存在与人类社会当中的，据联合国相关组织估计，在亚太地区，跨性别发生率约为千分之三，换算下来，我国约有四百万跨性别者。在学术界中，为什么有的人会是跨性别者一直是一个争论不休的话题，虽然讨论“跨性别的成因”是一个极具顺性别霸权视角的话题，但依然有相当数量的科学研究支持了跨性别是由先天因素导致且无法通过后天手段进行影响的。根据2011年发表的一项研究“人类大脑的性分化：与性别认同、性取向和神经精神疾病的关系<sup>1</sup>”中的结果表明，性别认同在出生前由大脑的性别划分决定，且大脑的性别划分由出生前的激素水平直接影响，与第一性征的划分无关，无法相互影响，且根据1995年发表的一项研究“人脑的性别差异及其与变性的关系<sup>2</sup>”中的结果表明，跨性别者的大脑结构更接近自己所认同的性别的大脑，而根据2019年发布的一项研究“跨性别儿童和顺性别儿童性别发展的相似性<sup>3</sup>”中的结果表明，跨性别者在儿童时期就表现出与其性别认同相关的表达，且没有证据表明出生后的成长经历可以影响性别认同。综上所述，我们可以得知一个人有何种性别认同是与生俱来的，而一个人的指派性别是否与其性别认同相符，决定了这个人是否是跨性别。我们需要明白，跨性别是处境而并非选择，跨性别无法被“成为”，一个人的性别认同与生俱来且伴随终生，人之所以具有多元性别认同就如同性取向的多样化一样，人人生来不同，但人人生而平等。需要注意的是，将“在后天成长经历中了解到跨性别相关概念后才发觉自己的跨性别身份”视为“认同是后天形成的”是一种完全偷换概念的做法，因担心有让人”成为跨性别“而避免让其接触相关信息，只会让跨性别者在不明不白中长大。跨性别是完全正常的现象，跨性别者相较于顺性别者，只有多数与少数之分，没有高低优劣之分，也更不存在“少数就是不正常”这种说法。

# 如何判断自己是否为跨性别/我是跨性别的情况吗？

我们可以通过以下条件和案例来判断自己为跨性别的可能性：

- 对自身的第一性征及其第二性征发育感到厌恶，希望掩盖或消除这些特征；
- 不希望具有指派性别的性别表达或希望具有与其相反的性别表达；
- 在他人用指派性别的称呼指代自己时感到不适并希望他人使用自己所认同的性别的称呼来指代自己；
- 对自己的指派性别和所给予的性别身份感到疑惑或不认同，不希望自己是这种性别或希望是另一种性别；
- 因自身的指派性别与性别认同不一致并因此产生了焦虑或抑郁状态，希望通过医疗手段，包括但不限于激素或手术治疗来改善等。

案例一：跨性别女性小兮在童年时期有了性别观念之后，发现自己的自我认同的性别与外界所期望的性别角色不符，因此感到十分困惑，不认为自己是男孩，不希望剪短发、想要穿裙子......在青春期时，第二性征的发展加剧了自我的不认同，对身体特征的变化如体毛发育和声音变粗等感到恶心，当被叫男性称呼时感到伤心，希望自知道自己是什么情况，因此开始自我探索、查找资料......在了解到“跨性别”这一概念后，结合了自己先前的经历与感受，确认自己属于跨性别的情况，欣慰于世界上还存在着和自己一样的人，寄希望于通过医疗，即激素和手术的方式“拯救”自己，并着手准备性别过渡。

当然，自己的情况与上述条件和案例并不意味着自己不是跨性别者，性别探索是一个长期且持续的过程，了解到跨性别相关概念之前没有过对自己的性别感到不认同也不能作为判断一个人不是跨性别者的依据。根据广义上的概念，只要符合“自己的指派性别与其性别认同不相符”即是跨性别者，并不强制要求一个人是否经历过性别焦虑。

# 作为跨性别者意味着什么/跨性别者如何面对家庭、社会与未来？

作为跨性别者，不可否认的一点是，相比普通人，我们会受到更多来自外界的不理解和针对，例如来自他人异的样眼光、辱骂、甚至是性别暴力。一个事实是，在官方层面和义务教育中，我过尚未对例如跨性别者这样的少数人群进行过客观性的知识普及，而顺性别者不会有与跨性别者相同的经历，也就不会去主动或从未了解过有关跨性别者的客观的信息，而有些甚至会获取到为博取流量、刻意污名化跨性别者的信息，再加之大多数人从未真正接触过跨性别者，直接导致了社会上的大多数人没有对跨性别群体甚至是“跨性别”、“性别认同”这种概念的的清晰认知，以此造成了跨性别者普遍不被接受与理解的局面，但这些并不意味着跨性别者无法获得幸福和成功。我们可以通过家庭宣教的方式，耐心的与家庭成员科普跨性别相关知识，让家长认识到，“跨性别”是一种处境而非选择，我们应当接纳自己与生俱来的性别认同，而非舍本逐末，强行扭转自己的性别认同，这种方式是不可取的，只会为我们带来更多的痛苦。而在社会中，跨性别者往往会面对更多挑战，其中较为常见的两个话题就是“医疗”与“转变”，在医疗方面，我们知道跨性别群体有着庞大的医疗需求，与其不对等的医疗资源及其获取难度与之形成了鲜明对比，而说起转变，则是性别肯定手术后绕不开的身份证件与学历信息的修改，在当前的政策下，跨性别者在完成手术后通过医院出具的相应材料修改身份证件上的性别信息，但在修改完身份信息后，如不在毕业前修改学籍信息，那么就会造成学历信息与本人不匹配的情况，而如果在毕业之后再修改身份信息，则又会出现学历信息不允许修改的情况，同样会导致“失去学历”，而跨性别者是否能在毕业之前进行手术，与家庭的支持密不可分，可以说是，对于有手术需求的跨性别者来说，家庭的支持与否直接决定了其在未来是否有能力在这个社会中生存下去。
